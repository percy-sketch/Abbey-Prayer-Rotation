Roster Settings

Auto-sync is on. The app checks for an updated roster.csv in your GitHub repository every time it opens.

How to update the roster

1. Get a fresh export from Realm (or have your admin prepare it)
2. Format it like the roster.csv template: First Name, Last Name, Member Status
3. Go to your GitHub repository → click roster.csv → click the pencil icon to edit, or upload a new file
4. Open the app — it will sync automatically
